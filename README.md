## 🚀 Projects

### 🗺️ [RoadSafe Analytics](https://github.com/Aman-2804/RoadSafe-Analytics)
Apache Spark pipeline that processes millions of NYC traffic collision records through a Bronze → Silver → Gold data lake. Produces a star-schema dimensional model, 19 pre-built SQL analytics queries, synthetic ICBC-aligned insurance claims, and interactive Plotly dashboards.  
`Apache Spark` `PySpark` `Docker` `Parquet` `Plotly`

### 🎬 [MovieMetric](https://github.com/Aman-2804/MovieMetric)
Distributed movie analytics backend that ingests data from TMDB, normalizes it in PostgreSQL, precomputes trending scores and genre stats via Celery background workers, and serves fast read-only REST APIs with Redis caching and Meilisearch full-text search. Includes CI/CD via GitHub Actions.  
`FastAPI` `PostgreSQL` `Redis` `Celery` `Meilisearch` `Docker`

### 🈳 [Manga Translator](https://github.com/Aman-2804/Manga-Translator)
End-to-end pipeline that takes raw manga pages (images or PDF) and returns fully translated versions — Japanese text erased with inpainting and replaced with translated text in the right font and size. Uses YOLOv8 for bubble detection, UNet++ for text segmentation, manga-ocr, and Claude Haiku for contextual translation.  
`Python` `YOLOv8` `UNet++` `FastAPI` `OpenCV` `Claude API`

### 🧠 [Emotion Detection in Song Lyrics (NLP)](https://github.com/Aman-2804/Emotion-Detection-in-Song-Lyrics-NLP)
Compared fine-tuned BERT against Qwen2-0.5B with in-context learning for 6-class emotion classification on GoEmotions. Fine-tuned BERT achieved **63.96% Macro F1**, outperforming vanilla BERT (10.18%) and Qwen2 few-shot prompting (11.18%). Includes attention visualization for interpretability.  
`Python` `PyTorch` `HuggingFace` `BERT` `NLP`
